API User Guide

The api reports location and speed of a specific satellite.

For source code, go to this github repository.

In testing, all SATELLITE will be defaulted to "ISS (ZARYA)" and all LON LAT ALT will be determined by location of ip the request is sent from

./api/get_time

gets current time

example json:
{"current_time": "2017-11-14T18:51:46"}

./api/get_velocity_vector[/SATELLITE]

gets velocity vector of SATELLITE in km/s
The origin at the center of mass of the earth, a point close to the Earth's center of figure
The Z axis on the line between the north and south poles, with positive values increasing northward (but does not exactly coincide with the Earth's rotational axis)[13]
The X and Y axes in the plane of the equator
The X axis passing through extending from 180 degrees longitude at the equator (negative) to 0 degrees longitude (prime meridian) at the equator (positive)
The Y axis passing through extending from 90 degrees west longitude at the equator (negative) to 90 degrees east longitude at the equator (positive)

example json:
{"velocity_vector_x": 4.142431217482311, "velocity_vector_y": -3.683046787698025, "velocity_vector_z": -5.296093950127022}

./api/get_velocity[/SATELLITE]

gets velocity of SATELLITE in km/s

example json:
{"velocity": 7.66929425407547}

./api/get_lonlatalt[/SATELLITE]

gets longitude latitude and altitude of SATELLITE

example json:
{"longitude": 55.11298676461255, "latitude": 39.95531405346363, "altitude": 406.60357716362483}

./api/get_az_el[/SATELLITE][/LON,LAT,ALT]

gets azimuth and elevation of SATELLITE at location (LON, LAT, ALT) in degrees

example json:
{"azimuth_angle": 159.5700144782178, "elevation_angle": -43.62036623984851}

MANUALLY UPDATE TLE

./api/update

updates TLE file used to compute all answers, recommend not use this function as the system automatically updates the file.
